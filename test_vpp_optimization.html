VPP Optimization Test

Open the browser console to see debug logs from the VPP optimization system.

What to look for in console:

  • DEBUG: processVPPInstancesOptimized called - Function is being triggered
  • DEBUG: VPP instances to process: X - Number of VPP chunks being processed
  • DEBUG: Available VPP classes - Shows which optimization functions are available
  • DEBUG: Applied geometry optimization - Geometry optimization working
  • DEBUG: Applied LOD optimization - LOD system working for distant chunks
  • DEBUG: VPP processing completed in X.XXms - Performance timing
